    constructor new Container 

    new Container(components[, options])
    • components (Array) – An array of Components that should initally be placed into the container. This is a required argument, however it can be an empty array. Also note that you can add or remove Components at any time via Container#addComponent and Container#removeComponent.
    • options (Object): The optional 'options' object's properties are copied 
                    this <code><a href="../../../components api/Container/index.html" title="Container (class)">Container</a></code> in the constructor. It allows all
              the same default properties as <code><a href="../../../components api/Component/index.html" title="Component (class)">Component</a></code>.

    Instantiates a new Container, adding the Components found in components initially.
